# J.A.R.V.I.S. Server — Инструкция

## 1. Установка Python
Скачай Python 3.10+ с https://python.org и установи.
При установке поставь галку "Add to PATH"!

## 2. Установка зависимостей
Открой командную строку (cmd) и выполни:

    pip install flask psutil

## 3. Настройка server.py
Открой server.py в блокноте и измени:
    STEAM_PATH      - путь к папке Steam (обычно уже правильный)
    EPIC_MANIFEST_PATH - путь к манифестам Epic (обычно уже правильный)

## 4. Запуск сервера
    python server.py

Сервер запустится на порту 5000. Не закрывай окно!

## 5. Узнать IP своего ПК
В cmd выполни:  ipconfig
Найди "IPv4-адрес" — это и есть твой IP (например 192.168.0.110)
Впиши его в приложении (main.dart, строка kIp)

## 6. Автозапуск (чтобы сервер стартовал с Windows НЕВИДИМО)

### Способ 1 — через планировщик задач (рекомендую):
1. Открой "Планировщик задач" (Task Scheduler)
2. Действие → Создать задачу
3. Общие → поставь "Выполнять для всех пользователей"
         → поставь "Выполнять с наивысшими правами"
4. Триггеры → Создать → "При входе в систему"
5. Действия → Создать:
     Программа:    pythonw.exe
     Аргументы:    C:\jarvis\server.py
     (замени путь на реальный)
6. Условия → снять "Только при питании от сети"
7. ОК → введи пароль Windows

Важно: pythonw.exe (с буквой w!) — запускает без окна консоли!

### Способ 2 — батник в автозагрузке:
1. Создай файл start_jarvis.bat с содержимым:
     @echo off
     start "" pythonw.exe C:\jarvis\server.py
2. Нажми Win+R → shell:startup → скопируй батник туда

## 7. Проверка
Открой браузер на телефоне (в той же сети Wi-Fi) и зайди на:
    http://192.168.0.110:5000/status
Если видишь JSON с данными — всё работает!

## Порты / Брандмауэр
Если не подключается — разреши порт 5000 в брандмауэре Windows:
    netsh advfirewall firewall add rule name="Jarvis" dir=in action=allow protocol=TCP localport=5000
